Web Application Development

What is Web Application Development?

This advanced vocational training course focuses on the development of web applications for different platforms and devices, where skills such as design, programming, databases, web security, user interface design, agile methodologies, and performance optimization are learned. Emerging technologies such as artificial intelligence and virtual reality are also studied to improve the user experience.

Multi-platform Application Development

What is Multi-platform Application Development?

In this advanced vocational training course, you learn to develop applications that work on various operating systems and devices, using different programming languages and tools. Topics such as mobile and web application development, multiplatform application design, databases, advanced features and software development methodologies are covered.

Computer Systems and Network Administration

What is Computer Systems and Network Administration?

This advanced vocational training course focuses on the administration and design of computer systems and networks. Skills such as network design and implementation, computer security, operating systems administration, data storage, virtualization, problem solving, and project management are learned. Additionally, knowledge about systems and networks infrastructure, business continuity, and agile methodologies are acquired.

Videogames

What is videogames vocational training course?

In this advanced vocational training course, you learn to create animation, game and interactive content using 3D modeling, animation, lighting and rendering tools, as well as to develop video games using game engines and programming tools. You learn to create characters, objects, visual and sound effects and to produce and publish games and interactive applications.
